通用问题

工具猫的产品免费吗？
21 款产品全部基于 MIT 协议 开源，个人和企业都可以免费使用、修改、再分发。无任何隐藏收费或功能限制。
需要什么运行环境？
Windows 10/11 系统。大部分工具提供可直接运行的 EXE 下载，少量基于 Python 的项目需要 Python 环境，但也在持续提供打包版本。
不会编程能用吗？
完全可以。大部分工具提供编译好的 EXE 可执行文件，下载解压后双击即可使用，无需任何编程知识或命令行操作。
会收集用户数据吗？
绝不。21 款产品全部 纯本地运行，不上传任何用户数据，无需联网即可使用，完全保护您的隐私安全。
在哪里下载？
在 GitHub 搜索 maomaozuiwudi 即可找到全部 21 个开源仓库。每个仓库的 Releases 页面提供 EXE 下载。
支持 Mac 或 Linux 吗？
目前以 Windows 平台为主。部分基于 Python 的项目理论上可在 Mac/Linux 上运行，但未做专门适配和测试。欢迎社区贡献跨平台支持。
有售后服务吗？
开源社区 + 公众号免费解答。欢迎在 GitHub 各仓库提交 Issue 反馈问题或建议。
怎么联系作者？
关注 小红书 @工具箱里的猫 获取最新动态，每天分享免费AI搜索技巧和效率工具。GitHub Issues 也是有效的反馈渠道。

PDF助手 （查看更多 →）

PDF助手有哪些功能？
包含 9 大功能：PDF 合并、PDF 拆分、OCR 文字识别、加密解密、文件压缩、添加水印、转图片、转 Word，覆盖日常 PDF 处理全场景。
PDF助手免费吗？有什么限制？
完全免费，MIT 开源协议。无功能限制、无水印、无使用次数限制、无文件大小限制，个人和企业均可免费使用。
PDF助手会泄露我的文件内容吗？
不会。PDF 助手 纯本地运行，所有处理都在你电脑上完成，文件不会被上传或联网传输，敏感文档可放心使用。
支持批量处理 PDF 吗？
支持。合并、拆分、压缩、转图片等功能均支持 批量操作，可一次选择多个文件进行处理，提高工作效率。
OCR 功能支持哪些语言？
支持 中英文混合识别，可准确识别扫描版 PDF 中的中文和英文字符，转为可编辑文本。
能处理加密的 PDF 吗？
如果知道密码，可以使用"解密"功能移除密码保护。加密功能则可以给 PDF 设置访问密码，保护文件安全。
PDF 转 Word 的排版效果如何？
尽量保留原始排版，包括字体、字号、段落结构等。但对于复杂排版（如多栏、表格），建议手动微调。

PluginGuard （查看更多 →）

PluginGuard 是什么？
Hermes Agent 插件安全检测工具。在安装插件前自动扫描，检测恶意代码、权限越界、隐私风险，是 AI 助手的「杀毒软件」。
PluginGuard 扫描哪些风险？
检测范围包括：恶意代码执行、文件系统越权访问（读取/写入敏感路径）、网络请求劫持、敏感信息窃取、环境变量泄漏等，生成详细的安全报告。
PluginGuard 对 Hermes 性能有影响吗？
影响极小。插件安装时触发一次扫描（通常 毫秒级），扫描结果缓存后不再重复检查，不影响正常使用性能。
误报如何处理？可以添加白名单吗？
支持白名单配置。如果确认某插件安全，可以将该插件加入白名单，之后将跳过检测。也欢迎提交 Issue 反馈误报情况。
PluginGuard 支持哪些插件格式？
支持目前 Hermes Agent 生态的 Python 插件，包括单文件 .py 插件和目录形式的插件包。
PluginGuard 本身安全吗？
PluginGuard 本身也遵循 MIT 协议开源，代码完全透明可审计。它只有读取权限，不会修改任何插件文件。
在哪里获取 PluginGuard？
GitHub 搜索 maomaozuiwudi/plugin-guard 即可找到，支持手动安装和自动集成到 Hermes 启动脚本。
