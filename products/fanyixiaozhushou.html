🌐 翻译小助手 常见问题

定义类

翻译小助手是什么？
翻译小助手（TransLens）是一款 Windows 桌面端翻译工具，支持划词翻译、翻译透镜（鼠标悬停即译）、OCR 图片文字识别翻译等多种模式。集成多家翻译引擎，纯本地+云端混合架构，满足日常阅读、文档翻译需求。
有哪些翻译模式？
翻译小助手提供三种核心模式：① 划词翻译 — 选中文字自动弹出翻译；② 翻译透镜 — 鼠标悬停即显示翻译结果，无需选中；③ OCR 翻译 — 截取屏幕区域，识别图片中的文字并翻译。三种模式可按需切换。
翻译透镜怎么用？
开启「翻译透镜」模式后，鼠标悬停在任意文字上（浏览器、文档、代码编辑器等），1 秒内自动弹出翻译气泡。支持自动检测源语言和切换目标语言，适合快速阅读外文资料。

实操类

划词翻译支持哪些软件？
划词翻译通过系统级热键实现，理论上支持所有 Windows 桌面软件：浏览器（Chrome/Edge/Firefox）、Office 套件（Word/Excel/PowerPoint）、PDF 阅读器、代码编辑器（VS Code/IDE）等。
支持哪些翻译引擎？
内置多家翻译引擎：DeepL、Google 翻译、微软翻译、百度翻译等。可在设置中自由切换默认引擎，也支持同时显示多个引擎结果进行对比。
翻译准确率怎么样？
翻译准确率主要取决于所选引擎。DeepL 在欧语互译方面表现优异；Google 和百度在中文互译方面成熟度高。翻译小助手本身不改变翻译质量，而是让你在多个引擎间灵活切换，取最准的结果。
OCR识别需要安装什么？
OCR 识别功能内置，无需额外安装。支持中、英、日、韩等常见语言的文字识别。截图后自动检测图片中的文字区域，识别完成后可直接复制或翻译。

对比类

需要联网吗？
翻译功能需联网调用翻译引擎，但划词截取和 OCR 识别本身在本地完成。如只使用本地 OCR 识别不翻译，可在离线环境正常工作。
需要API Key吗？
无需。翻译小助手内置了各翻译引擎的默认接入方式，开箱即用。如需使用个人 API Key 获得更高额度或速度，可在设置中配置。

商用类

免费吗？
翻译小助手对个人用户完全免费开放。内置翻译接口有每日合理使用限额，满足日常阅读需求。如需企业级高频使用，建议配置自己的翻译 API Key。
会泄露翻译内容吗？
翻译内容会经过所选翻译引擎的云端接口处理。如果你的翻译内容涉及隐私或商业机密，建议在设置中关闭「联网翻译」或使用本地部署的翻译模型。OCR 截图处理全程在本地完成，不会上传图片。
